Úvod

JI sdružuje několik užitečných knihoven.

Knihovny

  • Common - často používaná logika, struktury a výjimky
  • Čtení/zápis souborů - práce s JSONem a factory pro BufferedReader/Writter
  • Lokalizace - pro vícejazyčné aplikace
  • Databaze - Komunikace s databází, query builder, migrace
  • Testování vůči databázi
  • Komunikace klient-server - web server, Rest API requesty, P2P komunikace

Závislost

JI lzde do projektu začlenit celé, nebo jen vybrané části.

JI je publikováno na platformě JitPack. Ta umožňuje přidat projekt jako Gradle nebo Maven závislost.

Nejprve je nutné do repositories přidat:

maven { url 'https://jitpack.io' }
