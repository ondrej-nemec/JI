Struktury

Dictionary

Skupina tříd označených jako 'Dictionary' pomáhá pracovat s objekty, u kterých není známý datový typ, jedná se o vstupní data s proměnným typem, nebo jsou data ve stringu.

DictionaryValue

Wrapper pro hodnotu. Umožňuje nastavit podle jakého vzoru se má parsovat datum a čas. Taktéž lze nastavit funkci, která převede text na Map nebo List. Ve výchozím nastavení se pokusí najít třídu pro parsování JSONu a zkusí text zparsovat.

ListDictionary

Nedědí od List, lze na List převést. Pro DictionaryValue je ListDictionary totožný s List (automaticky převede)

MapDictionary

Nedědí od Map, lze na Map převést. Pro DictionaryValue je MapDictionary totožný s Map (automaticky převede)

Podporované parsování

  • Boolean, boolean - objekt je převeden na String, pokud hodnota stringu je "true" nebo "on" nebo "1", vrací true
  • Čísla (včetně primitivních typů): Byte, Short, Integer, Long, Float, Double
  • Text: Character, char, String
  • Map, MapDictionary - Dictionary převání jedno v druhé a zpět automaticky podle potřeby
  • List, ListDictionary - Dictionary převání jedno v druhé a zpět automaticky podle potřeby. Případně lze zístat List ze Stringu rozdělením daným textem (String::split).
  • Enumy
  • Čas: LocalDate, LocalTime, LocalDateTime, ZonedDateTime
  • Libovolnou entitu
