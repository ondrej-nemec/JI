About Files package

JI Files simplifies reading and writing to files. Especialy provide factories for creating BufferedReader and BufferedWritter. It is able to parse from and to JSON, too.

Dependency

See JitPack or you use Gradle:

gradle.build


repositories { 
	...
	maven { url 'https://jitpack.io' }
}

dependencies {
    implementation 'com.github.ondrej-nemec:JI:ji-files:Tag'
}
