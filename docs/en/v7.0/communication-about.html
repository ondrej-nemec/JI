About Client-Server

This package helps easy establish (secured or unsecured) socket connection. Can be used for P2P communication, as HTTP server or making Rest API requests.

Dependency

See JitPack or you use Gradle:

gradle.build


repositories {
	...
	maven { url 'https://jitpack.io' }
}
dependencies {
	implementation 'com.github.ondrej-nemec:JI:ji-communication:Tag'
}
