About TOTI

Project contains some useful libraries, functions and structures

Libraries

  • Common - Often used logic, useful structures and exceptions
  • File reading/writing - Factories for Buffered Reader/Writer and JSON stream
  • Lokalization - Package supports multilingual applications./li>
  • Database - Contains: database communication, SQL Query Builder and database migrations.
  • Database Test Case - Extends jUnit for testing with database
  • Client-Server communication - Provide server/client communication (secured or unsecured) using Java sockets. Implements web server too.

Dependency
